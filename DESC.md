# 🎯 Gap-time Companion Agent
## AI 기반 맞춤형 여행 추천 서비스

### 🌟 서비스 개요
**Gap-time Companion Agent**는 사용자의 짧은 여유 시간을 최대한 활용할 수 있도록 AI가 맞춤형 여행 추천을 제공하는 혁신적인 서비스입니다. 단순한 장소 추천을 넘어서, 사용자의 현재 상황(위치, 날씨, 시간)과 개인적 선호도를 종합적으로 고려하여 최적의 활동을 제안합니다.

---

## 🚀 핵심 기능

### 1. **지능형 컨텍스트 분석**
- **실시간 위치 기반**: 사용자의 현재 위치를 정확히 파악하여 근처 최적 장소 추천
- **날씨 연동**: 현재 날씨와 온도를 고려한 실내/실외 활동 구분 추천
- **시간 제약 고려**: 사용 가능한 시간에 맞는 활동만 선별하여 제안

### 2. **AI 맞춤형 질의응답 시스템**
- **동적 질문 생성**: LLM이 사용자의 상황과 선호도를 바탕으로 개인화된 질문 자동 생성
- **맞춤형 상호작용**: 고정된 질문이 아닌, 사용자의 답변에 따라 다음 질문이 동적으로 변화
- **깊이 있는 선호도 파악**: 단순한 선택지가 아닌 자연어 대화를 통한 세밀한 니즈 분석

### 3. **다층적 추천 알고리즘**
- **다양한 검색 소스**: Google Maps, SerpAPI 등 여러 검색 엔진을 활용한 포괄적 장소 검색
- **지능형 필터링**: 이동 시간, 예산, 테마별 다차원 필터링으로 정확한 추천
- **LLM 기반 최종 선별**: AI가 사용자의 답변과 상황을 종합하여 최적의 4개 장소 선별

### 4. **실시간 정보 통합**
- **실제 리뷰 수집**: Google Maps에서 실시간 리뷰와 평점 정보 수집
- **사진 제공**: 각 장소의 실제 사진을 제공하여 시각적 정보 지원
- **정확한 길찾기**: Google Maps 연동으로 정확한 경로와 소요 시간 제공

## 💡 서비스의 차별화 포인트

### 🎯 **상황 인식형 추천**
- 단순한 "인기 장소"가 아닌, **현재 상황에 최적화된** 장소 추천
- 날씨가 흐리면 실내 활동, 맑으면 야외 활동을 자동으로 고려
- 사용자의 현재 위치에서 도보 가능한 거리 내 장소만 선별

### 🧠 **AI 기반 개인화**
- 사용자마다 다른 질문을 통해 **진정한 개인화** 실현
- "쇼핑"이라는 단순한 답변이 아닌, "현지 감성의 핸드메이드 기념품" 같은 구체적 니즈 파악
- 대화형 인터페이스로 사용자의 숨겨진 선호도까지 발굴

### ⚡ **시간 효율성 극대화**
- 사용 가능한 시간을 정확히 계산하여 **과도하거나 부족한 시간 배정 방지**
- 이동 시간, 체류 시간, 대기 시간을 모두 고려한 현실적 일정 제안
- 30분, 1시간 등 짧은 시간도 의미 있게 활용할 수 있는 활동 제안

### 🔄 **실시간 정보 반영**
- 정적 데이터가 아닌 **실시간 리뷰와 평점** 기반 추천
- 최신 사진과 실제 방문자 후기를 통한 신뢰성 있는 정보 제공
- 계절, 시간대, 날씨 등 변동 요소를 실시간으로 반영

## 🎨 사용자 경험

### **직관적인 3단계 프로세스**
1. **기본 정보 입력**: 시간, 예산, 테마 등 간단한 선택
2. **AI 맞춤 질문**: 개인화된 질문에 대한 자연스러운 답변
3. **맞춤 추천**: 상황에 최적화된 4개 장소와 상세 정보 제공

### **풍부한 정보 제공**
- 각 장소별 상세 리뷰 요약 및 실제 리뷰 3개
- 고품질 사진 갤러리
- 정확한 이동 시간 (도보/차량/대중교통)
- 예상 비용 정보
- Google Maps 연동 길찾기

### **관광객**
- 제한된 시간 내 최적의 관광 경험
- 날씨와 상황에 맞는 유연한 일정 조정

### **현지인**
- 평소 놓치기 쉬운 근처 숨은 명소 발견
- 새로운 경험을 위한 맞춤형 추천

## 🏆 서비스의 가치

### **개인화의 진화**
- "많은 사람이 좋아하는" 장소가 아닌 **"나에게 딱 맞는"** 장소 추천
- AI가 사용자의 답변을 분석하여 숨겨진 선호도까지 파악

### **시간의 가치 극대화**
- 제한된 시간을 가장 의미 있게 활용할 수 있는 방법 제시
- 단순한 시간 소비가 아닌 **가치 있는 경험** 제공

### **지역 경제 활성화**
- 숨겨진 현지 비즈니스와 문화 공간 발굴
- 관광객과 현지인 모두에게 새로운 발견의 기회 제공
